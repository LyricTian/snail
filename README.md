# Snail
> 文件下载服务

![screenshot](https://raw.githubusercontent.com/LyricTian/snail/master/static/img/screenshot.png)

## 获取

``` bash
$ go get -u github.com/LyricTian/snail
```

## 安装并运行

``` bash
$ cd $GOPATH/src/github.com/LyricTian/snail
```

### 安装js依赖

> 需要先安装[nodejs](https://nodejs.org)

``` bash
$ npm install -g bower
$ bower install
```

## 运行

> 配置参数(conf/app.conf)

``` bash
$ bee run
```
